.. 产品使用注意事项:

=====================================
第二章  产品使用注意事项
=====================================

     .. image:: ../images/right.png

   1.产品上电前请务必保证天线完全安装，以免上电造成模块损坏。

       .. image:: ../images/voltage.png

   2.电源电压切勿接反，以免烧坏设。

       .. image:: ../images/care.png
       
   3.请保持模块干燥，勿靠近水。
   
   4.请小心保存模块，勿摔，勿撞，勿跌落。
