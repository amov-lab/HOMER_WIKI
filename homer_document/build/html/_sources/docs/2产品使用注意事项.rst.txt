.. 产品使用注意事项:

=====================================
第二章  产品使用注意事项
=====================================

一、在使用时避免环境外部因素影响，影响系统的正常使用，以下三个因素在使用中对无线图传系统影响最大，使用时应避免：
1.传输距离。无线图传系统的传输距离有有大有小，但是距离的远近就会影响到信号的强弱问题。无线电波由于自身的物理特性，存在着衰减问题。因此，当距离无线发射源过远时，信号就会变得很弱，而当距离无线发射源越近，信号强度就会越强。
2.障碍物。无线图传信号在穿越钢筋混凝土墙、钢板、 树木、金属制品等障碍物后，信号会大幅度衰减。很多人在使用时隔墙传输时信号会大大减弱，其实产品本身没有问题，而是这些障碍物让传输信号减弱了，所以就会出现信号中断等一系列问题。在使用中建议用户尽量避开-些障碍物，在室内使用最好将无线图传放在靠近窗户或[门的位置，这样可以减少墙体对无线信号的阻隔。

3.信号干扰。在电磁环境、强信号、同频率无线设备等都会对无线传输信号产生较强的干扰，所以在使用无线专输时尽量避开对传输信号影响较大的一些环境中。
二、人为因素也有可能影响到使用。比如:安装错误、设备在使用中跌落等原因。有以下注意事项：
1．ZGET HOMER输入电压：8V-30V，切勿将正负极接错，导致模块永久损坏。
2.确保所有天线，电源，串口，网口接口连接可靠以及正确。
3. ZGET HOMER工作频段为5.1GHz~5.9GHz频段，发射端和接收端在同一个频点上，即可相互接收，多个频点可以切换。

三、环境要求
1．温度。模块避免高温环境造成氧化等化学反应，造成绝缘结构，表面防护层迅速老化，速被破坏；增强水汽的穿透能力和水汽的破坏能力；使有些物质软化、融化，使结构在机械应力下损坏；使润滑剂粘度减小和蒸发，丧失润滑能力；使物体发生膨胀变形，从而导致机械应力加大，运行零件磨损增大或结构损坏；对于发热量大的电子产品来说，高温环境会使机内温度上升到危险程度，使电子元器件损坏或加速老化，使用寿命大大缩短。
2．湿度。湿度也是环境中起重大作用的一个因素，特别是它和温度因素结合在一起时，往往会产生更大的破坏作用。高湿度使物理性能下降、绝缘电阻降低、介电常数增加、机械强度下降，以及产生腐蚀、生锈和润滑油劣化等。无论在电子产品使用状态或运输保管状态都会引起这些问题。相反，干燥会引起干裂与脆化，使机械强度下降，结构失效及电气性能发生变化。
3．气压。气压降低、空气稀薄所造成的影响主要有：散热条件差、空气绝缘强度下降、灭弧困难。气压主要随海拔的增加而按指数规律降低。
